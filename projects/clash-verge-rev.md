## 项目简介

Clash Verge Rev 是基于 Tauri 框架开发的 Clash Meta GUI 客户端，是 Clash Verge 项目的延续版本。

## 主要特性

- **跨平台支持**: 支持 Windows (x64/x86)、Linux (x64/arm64) 和 macOS 10.15+ (Intel/Apple Silicon)
- **高性能**: 基于 Rust 和 Tauri 2 框架构建，性能强劲
- **内置内核**: 内置 Clash.Meta(mihomo) 内核，支持切换 Alpha 版本内核
- **界面美观**: 简洁美观的用户界面，支持自定义主题颜色、代理组/托盘图标以及 CSS 注入
- **配置管理**: 配置文件管理和增强（Merge 和 Script），配置文件语法提示
- **系统代理**: 系统代理和守卫、TUN(虚拟网卡)模式
- **可视化编辑**: 可视化节点和规则编辑
- **配置备份**: WebDav 配置备份和同步

## 技术栈

- **框架**: Tauri 2
- **语言**: Rust、TypeScript
- **内核**: Clash.Meta (mihomo)
- **前端**: Vite
