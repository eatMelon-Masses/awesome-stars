# awesome-stars

> 用于收藏和备注你喜欢的 GitHub 项目

## 使用方法

1. 每当你发现一个有趣的项目，在本仓库创建一个 Issue 或 Discussion。
2. 在 Issue 标题填写项目名称或用途，如 `GoMusic - 音乐播放器学习`。
3. 在 Issue 内容中，添加：
   - 项目链接（如：https://github.com/Bistutu/GoMusic）
   - 你的备注（用途、亮点、待学习内容等）
   - 推荐标签（如：`前端`、`爬虫`、`AI`）

## 推荐分类标签

- 前端
- 后端
- 工具
- 学习
- AI
- 数据分析
- 爬虫
- 其他

## 示例 Issue 内容

```
项目：GoMusic
链接：https://github.com/Bistutu/GoMusic
标签：音乐、学习
备注：用 Go 开发的音乐播放器，适合学习 Go 的项目结构和音频处理。
```

## 其他建议

- 可以用 GitHub Label 为 Issue 分类，方便检索。
- 如果内容较多，可以用 Discussion 代替 Issue，支持更丰富交流和补充。

---
本仓库仅用于个人收藏和知识管理，不存放项目代码。
